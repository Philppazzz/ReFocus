# ReFocus

**Screen-Locking Android Application Using Decision Tree for Managing Screen-Time Overuse**

ReFocus is an academic thesis project that aims to help users build healthier digital habits by intelligently monitoring and managing their smartphone usage.  
The app uses **Artificial Intelligence (Decision Tree Machine Learning model)** to detect overuse patterns and automatically control access to distracting apps like games and social media.
